☁️ Microsoft 365 Adminliği

🎯 Microsoft 365 Neden Önemli?

Microsoft 365, modern işletmelerin dijital dönüşümünün anahtarıdır. Neden Microsoft 365 kullanmalıyız?

🔍 Microsoft 365'in Faydaları:

  • Bulut Avantajları: Her yerden erişim, otomatik güncellemeler
  • Maliyet Tasarrufu: On-premises sunucu maliyetleri yok
  • İşbirliği: Ekip çalışması için gelişmiş araçlar
  • Güvenlik: Enterprise seviye güvenlik özellikleri
  • Ölçeklenebilirlik: Kullanıcı sayısına göre esnek lisanslama

🏢 Gerçek Hayat Senaryosu:

Problem: 300 çalışanlı şirkette on-premises sistemler yetersiz

Sonuç: Yavaş performans, yüksek maliyet, güvenlik açıkları

Çözüm: Microsoft 365 ile bulut geçişi

👤 Kullanıcı Yönetimi

Kullanıcı Oluşturma

# PowerShell ile kullanıcı oluşturma
Connect-MsolService

# Yeni kullanıcı oluştur
New-MsolUser -UserPrincipalName "john.doe@company.com" `
    -DisplayName "John Doe" `
    -FirstName "John" `
    -LastName "Doe" `
    -Password "TempPass123!" `
    -ForceChangePassword $true

# Lisans atama
Set-MsolUserLicense -UserPrincipalName "john.doe@company.com" `
    -AddLicenses "company:ENTERPRISEPACK"

# Kullanıcı grubuna ekleme
Add-MsolGroupMember -GroupObjectId "group-guid" `
    -GroupMemberType "User" `
    -GroupMemberObjectId "user-guid"

Toplu Kullanıcı Yönetimi

# CSV dosyasından toplu kullanıcı oluşturma
$users = Import-Csv -Path "C:\Users\new_users.csv"

foreach ($user in $users) {
    # Kullanıcı oluştur
    New-MsolUser -UserPrincipalName $user.Email `
        -DisplayName $user.FullName `
        -FirstName $user.FirstName `
        -LastName $user.LastName `
        -Password "TempPass123!" `
        -ForceChangePassword $true
    
    # Lisans ata
    Set-MsolUserLicense -UserPrincipalName $user.Email `
        -AddLicenses "company:ENTERPRISEPACK"
    
    Write-Host "Kullanıcı oluşturuldu: $($user.FullName)" -ForegroundColor Green
}

📧 Exchange Online Yönetimi

Mailbox Yönetimi

# Exchange Online'a bağlan
Connect-ExchangeOnline

# Mailbox oluşturma
New-Mailbox -UserPrincipalName "john.doe@company.com" `
    -Alias "john.doe" `
    -Name "John Doe" `
    -DisplayName "John Doe" `
    -Password (ConvertTo-SecureString "TempPass123!" -AsPlainText -Force)

# Mailbox özelliklerini güncelleme
Set-Mailbox -Identity "john.doe@company.com" `
    -ProhibitSendQuota 2GB `
    -ProhibitSendReceiveQuota 2.5GB `
    -IssueWarningQuota 1.5GB

# Distribution group oluşturma
New-DistributionGroup -Name "IT Department" `
    -DisplayName "IT Department" `
    -Alias "IT-Dept" `
    -ManagedBy "admin@company.com"

# Distribution group'a üye ekleme
Add-DistributionGroupMember -Identity "IT Department" `
    -Member "john.doe@company.com"

Mail Flow Kuralları

# Mail flow kuralı oluşturma
New-TransportRule -Name "Block External Attachments" `
    -FromScope "NotInOrganization" `
    -AttachmentHasExecutableContent $true `
    -RejectMessageReasonText "Executable attachments are not allowed from external senders"

# Spam filtreleme kuralı
New-TransportRule -Name "High Confidence Spam" `
    -SetHeaderName "X-Spam-Level" `
    -SetHeaderValue "High" `
    -SclValue 5 `
    -Quarantine $true

📁 SharePoint Online Yönetimi

Site Collection Oluşturma

# SharePoint Online'a bağlan
Connect-PnPOnline -Url "https://company.sharepoint.com" -Interactive

# Site collection oluşturma
New-PnPSite -Type TeamSite -Title "IT Department" `
    -Alias "it-dept" `
    -Description "IT Department collaboration site"

# Site izinleri ayarlama
Set-PnPWebPermission -Identity "IT Department" `
    -User "john.doe@company.com" `
    -AddRole "Full Control"

# List oluşturma
New-PnPList -Title "Project Documents" `
    -Template DocumentLibrary `
    -OnQuickLaunch

Doküman Yönetimi

# Dosya yükleme
Add-PnPFile -Path "C:\Documents\project.docx" `
    -Folder "Shared Documents" `
    -NewFileName "Project Plan.docx"

# Dosya izinleri ayarlama
Set-PnPListItemPermission -List "Documents" `
    -Identity 1 `
    -User "john.doe@company.com" `
    -AddRole "Contribute"

# Versioning etkinleştirme
Set-PnPList -Identity "Documents" `
    -EnableVersioning $true `
    -MajorVersions 10

💬 Microsoft Teams Yönetimi

Team Oluşturma

# Teams'e bağlan
Connect-MicrosoftTeams

# Team oluşturma
New-Team -DisplayName "IT Department" `
    -Description "IT Department collaboration team" `
    -Visibility "Private"

# Team'e üye ekleme
Add-TeamUser -GroupId "team-guid" `
    -User "john.doe@company.com" `
    -Role "Member"

# Channel oluşturma
New-TeamChannel -GroupId "team-guid" `
    -DisplayName "General Discussion" `
    -Description "General team discussions"

Teams Politikaları

# Teams meeting politikası oluşturma
New-CsTeamsMeetingPolicy -Identity "RestrictedMeetings" `
    -AllowIPVideo $false `
    -AllowAnonymousUsersToStartMeeting $false `
    -AllowAnonymousUsersToJoinMeeting $false

# Teams messaging politikası
New-CsTeamsMessagingPolicy -Identity "RestrictedMessaging" `
    -AllowUserChat $true `
    -AllowUserDeleteMessage $false `
    -AllowUserEditMessage $false

🔒 Güvenlik ve Uyumluluk

Multi-Factor Authentication

# MFA'yı etkinleştir
Set-MsolUser -UserPrincipalName "john.doe@company.com" `
    -StrongAuthenticationRequired $true

# MFA durumunu kontrol et
Get-MsolUser -UserPrincipalName "john.doe@company.com" `
    | Select-Object UserPrincipalName, StrongAuthenticationRequirements

Conditional Access

# Conditional Access politikası oluşturma
New-AzureADMSConditionalAccessPolicy -DisplayName "Require MFA for Admin" `
    -State "Enabled" `
    -Conditions @{
        Users = @{
            IncludeUsers = @("admin@company.com")
        }
        Applications = @{
            IncludeApplications = @("All")
        }
        ClientAppTypes = @("Browser", "MobileAppsAndDesktopClients")
    } `
    -GrantControls @{
        BuiltInControls = @("Mfa")
    }

Data Loss Prevention (DLP)

# DLP politikası oluşturma
New-DlpCompliancePolicy -Name "Financial Data Protection" `
    -Comment "Protect financial data from unauthorized sharing" `
    -Mode "Enforce"

# DLP kuralı oluşturma
New-DlpComplianceRule -Name "Credit Card Detection" `
    -Policy "Financial Data Protection" `
    -ContentContainsSensitiveInformation @{
        SensitiveInformation = @{
            Name = "Credit Card Number"
            MinCount = 1
        }
    } `
    -BlockAccess $true

📊 İzleme ve Raporlama

Kullanım Raporları

# Office 365 kullanım raporları
Get-Report -ReportName "Office365ActiveUsers" `
    -StartDate (Get-Date).AddDays(-30) `
    -EndDate (Get-Date)

# Exchange Online raporları
Get-MailboxStatistics -Identity "john.doe@company.com" `
    | Select-Object DisplayName, ItemCount, TotalItemSize

# Teams kullanım raporları
Get-TeamUser -GroupId "team-guid" `
    | Select-Object User, Role, LastActivityDate

Güvenlik Raporları

# Güvenlik olayları
Get-AuditLog -StartDate (Get-Date).AddDays(-7) `
    -EndDate (Get-Date) `
    -Operations "UserLoggedIn"

# Riskli kullanıcılar
Get-AzureADUserRiskPolicy `
    | Select-Object DisplayName, RiskLevel

# MFA durumu
Get-MsolUser -All `
    | Where-Object {$_.StrongAuthenticationRequirements -ne $null} `
    | Select-Object UserPrincipalName, StrongAuthenticationRequirements
