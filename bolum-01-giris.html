📚 Giriş ve Temel Kavramlar

🎯 IT Dünyasına Hoş Geldiniz

Bu eğitim programı, IT dünyasında kariyer yapmak isteyenler için kapsamlı bir rehber niteliğindedir. Sistem yöneticiliği, ağ yönetimi, güvenlik ve modern IT teknolojileri konularında derinlemesine bilgi sahibi olacaksınız.

🎓 Öğrenme Hedefleri

  • IT alanında temel kavramları öğrenmek
  • Sistem yöneticiliği rollerini anlamak
  • Ağ teknolojilerinin temellerini kavramak
  • Güvenlik prensiplerini öğrenmek
  • Modern IT araçlarını kullanabilmek

🚀 Kariyer Yolları

🖥️ Sistem Yöneticisi

Sunucu yönetimi, kullanıcı hesapları, backup ve restore işlemleri

  • Windows Server yönetimi
  • Active Directory yapılandırması
  • PowerShell scripting

🌐 Ağ Yöneticisi

Ağ altyapısı tasarımı, güvenlik duvarı yönetimi, ağ performansı

  • TCP/IP protokolleri
  • Router ve switch yönetimi
  • VPN yapılandırması

🔒 Siber Güvenlik Uzmanı

Güvenlik tehditlerini analiz etme, güvenlik politikaları oluşturma

  • Penetration testing
  • Incident response
  • Security monitoring

☁️ Cloud Architect

Bulut altyapısı tasarımı, hibrit çözümler, DevOps süreçleri

  • Azure/AWS yönetimi
  • Container teknolojileri
  • Infrastructure as Code

🔧 IT Temel Kavramları

💻 Donanım (Hardware)

Sunucu Donanımları

  • CPU: İşlemci gücü, çekirdek sayısı
  • RAM: Bellek kapasitesi, hızı
  • Storage: SSD/HDD, RAID yapılandırması
  • Network: Ağ kartları, bant genişliği

🌐 Ağ Temelleri

OSI Modeli

  1. Physical: Fiziksel bağlantı
  2. Data Link: MAC adresleri
  3. Network: IP adresleri, routing
  4. Transport: TCP/UDP protokolleri
  5. Session: Oturum yönetimi
  6. Presentation: Veri formatı
  7. Application: Uygulama katmanı

🔒 Güvenlik Prensipleri

CIA Triad

  • Confidentiality (Gizlilik): Yetkisiz erişimi engelleme
  • Integrity (Bütünlük): Veri değişikliklerini kontrol etme
  • Availability (Erişilebilirlik): Sürekli hizmet sağlama

🏢 Gerçek Hayat Senaryosu

Orta Ölçekli Şirket IT Altyapısı

Şirket: 500 çalışanlı teknoloji firması

IT Altyapısı:

  • 2 adet Domain Controller (Windows Server 2019)
  • 1 adet Exchange Server (Mail sistemi)
  • 1 adet File Server (Dosya paylaşımı)
  • FortiGate Firewall (Güvenlik duvarı)
  • Microsoft 365 (Office uygulamaları)

Günlük IT Operasyonları:

  1. Sabah 08:00: Sistem sağlık kontrolü
  2. 09:00-17:00: Kullanıcı destek talepleri
  3. 17:00-18:00: Backup işlemleri
  4. Gece: Sistem güncellemeleri
